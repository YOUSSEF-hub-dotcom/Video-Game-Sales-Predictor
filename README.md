# Video-Game-Sales-Predictor
## Overview
This Project aims to predict  **Global Sales** of video games using machine learning models , while providing visualization to analyze market trends in the gaming industry.
# Methodology
1. **Data Preprocessing**:Cleaned Dataset, handeld missing values, and encoded categorical variables.
2. **Exploratory Data Analysis (EDA)**: visualized sales trends by genre, platform, and region using Matplotlib and Seaborn.
3. **Modeling**:
4. 1. **Linear Regression**:Acheved R^2 Score = 0.9999999, including a near-perfect but potential overfitting.
   2. **Decision Tree Regressor**:Applied to capture non-linear patterns and patterns and reduce overvitting.
   3. **Randon Forest Regressor**:Used for robust ensamble predictions.
5. **Evaluations**:
6. Compared models using R^2 Score , Mean squared Error (MSE), Mean absolute error (MAE), and model.score.
## Results
7.Linear Regression showed near-perfect fit(R^2=0.999999),but Decision Tree Random Forest provided better generalization.
